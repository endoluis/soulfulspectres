
# Documentación Interna - Soulful Spectres Website

Este documento describe brevemente la función de cada archivo incluido en el paquete comprimido del sitio web.

---

## Archivos HTML
- **index.html**: Página principal del sitio. Contiene la estructura del laberinto inicial y enlaces al tríptico de origen.
- **inicio.html**: Página secundaria, posiblemente usada como alternativa o entrada narrativa.
- **origen.html**: Página de desarrollo de los orígenes conceptuales o artísticos del proyecto.

## Archivos CSS
- **style.css**: Hoja de estilo principal aplicada al index.
- **styles.css**: Variante o extensión de estilos, podría ser redundante o contener ajustes alternativos.
- **estilo.css**: Posible hoja de estilo enfocada a estructura narrativa o visual alternativa. Necesita verificación de uso activo.

## Archivos JavaScript
- **script.js**: Script principal del sitio (interactividad básica o navegación).
- **keys.js**: Control de acceso por claves al sistema laberíntico (fase avanzada o futura del proyecto).

## Archivos de configuración SEO
- **robots.txt**: Indica a los buscadores cómo indexar o no el contenido del sitio.
- **sitemap.xml**: Mapa del sitio estructurado para facilitar el rastreo e indexación por motores de búsqueda.

## Archivos de documentación y control
- **README.txt**: Texto de referencia (no estructurado).
- **README.md**: Versión Markdown del texto anterior, para lectura clara en plataformas tipo GitHub.
- **CNAME.txt**: Archivo que asocia el dominio personalizado `soulfulspectres.com` al repositorio en GitHub Pages.
